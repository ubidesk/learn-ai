# Curriculum Quality Standards

Every lesson merged into Learn AI must clear these bars. This is the
checklist a reviewer runs against a pull request before approving new
curriculum content.

## Correctness

- Technically accurate at the level of an expert practitioner.
- No sloppy analogies that will need to be un-taught later.
- Claims about industry, tooling, or research are cited or explicitly
  marked as opinion.

## Pedagogical soundness

- Prerequisites are declared and are all taught earlier in Learn AI.
- Introduces at most one hard new idea.
- Follows the Learn → Visualize → Practice → Build → Reflect rhythm.
- Includes at least one retrieval check and one applied exercise.
- Provides a clear, learner-visible signal of understanding.

## Writing quality

- Written for a smart beginner, not a peer expert.
- Short sentences. Concrete nouns. Active voice.
- Terminology is introduced with a definition the first time it is used.
- No filler, no hype, no cleverness for its own sake.
- Reading level and cognitive load are appropriate for stage.

## Accessibility

- Semantic headings, meaningful alt text, keyboard-operable interactives.
- Sufficient color contrast; never color-only signaling.
- No auto-playing motion; respects `prefers-reduced-motion`.
- Code samples are copyable and screen-reader friendly.

## Honesty

- Names what is hard, what is optional, and what is hype.
- Distinguishes "the truth", "a useful simplification", and "a lie we
  will fix later".
- Estimates of duration and difficulty are realistic for a true beginner.

## Coherence with the whole

- Uses the same terms, notation, and mental models as neighboring
  lessons.
- Reuses earlier examples where possible instead of inventing new ones.
- Sets up, or pays off, adjacent lessons deliberately.

## Reviewer checklist

- [ ] A complete beginner can follow this without external help.
- [ ] An expert would not wince at any claim made here.
- [ ] The lesson objective is testable, and the practice tests it.
- [ ] Nothing here needs to be un-taught later.
- [ ] Removing this lesson would leave a real gap in the curriculum.

A lesson that cannot check every box is not ready to merge.
